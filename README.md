## 📖 Overview
**countingdeaths** is a lightweight server-side plugin designed for **Minecraft 1.21.10**. It automates the tracking of player deaths using native statistics and presents them through a clean, real-time scoreboard.

## 🚀 Installation
1. Navigate to your server's `plugins/` directory.
2. Drop the compiled `countingdeaths.jar` into the folder.
3. Restart your server to enable the utility.

## 🛠️ Developer Setup (Cloning)
If you want to modify the code or contribute to the project, follow these steps to set up the workspace on your machine:

### 1. Clone the Repository
Open your terminal or Git Bash and run:
```bash
git clone https://github.com/YanirValdman/countingdeaths.git
```
2. Import to IntelliJ IDEA
Launch IntelliJ IDEA and select Open.

Navigate to the folder where you cloned the project and click OK.

Ensure you have the Spigot API (1.21.10) added as a library in your Project Structure (Ctrl + Alt + Shift + S).

3. Building the JAR
Go to File > Project Structure > Artifacts.

Add a new JAR from modules with dependencies.

Use Build > Build Artifacts to generate your own version of the plugin.
